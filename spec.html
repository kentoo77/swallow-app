⚡ 概要

Swallow コンテンツパイプラインは、先生が撮影した動画1本から、英会話レッスンカードを全自動で生成するシステムです。

10分の撮影 → 20〜30レッスンカード自動生成。月100フレーズ量産しても月75円。

2️⃣ Step 2: Whisper API で文字起こし 健人

動画から音声を抽出し、Whisper APIに送信。英語フレーズごとにタイムスタンプ付きで返ってくる。

処理内容

  • ffmpegで動画 → WAV音声に変換（16kHz モノラル）
  • Whisper APIに送信（language="en" で英語のみ認識）
  • 日本語（「えーっと」「次は」等）は自動的に除外
  • 3単語未満の短いフレーズもフィルタ

コード: 音声抽出 + 文字起こし

pythonimport json, subprocess, os from openai import OpenAI from config import OPENAI_API_KEY, MIN_PHRASE_LENGTH client = OpenAI(api_key=OPENAI_API_KEY) def extract_audio(video_path: str, audio_path: str): """動画から音声を抽出（Whisper API用）""" subprocess.run([ "ffmpeg", "-i", video_path, "-vn", # 映像なし "-acodec", "pcm_s16le", # WAV形式 "-ar", "16000", # 16kHz（Whisper推奨） "-ac", "1", # モノラル "-y", audio_path ], check=True, capture_output=True) def transcribe(audio_path: str) -> list[dict]: """Whisper API で文字起こし（タイムスタンプ付き）""" with open(audio_path, "rb") as f: response = client.audio.transcriptions.create( model="whisper-1", file=f, language="en", response_format="verbose_json", timestamp_granularities=["segment"] ) phrases = [] for segment in response.segments: text = segment.text.strip() if len(text.split()) < MIN_PHRASE_LENGTH: continue if any('\u3040' <= c <= '\u9fff' for c in text): continue phrases.append({ "text": text, "start": round(segment.start, 2), "end": round(segment.end, 2), }) return phrases

Whisper API の戻り値イメージ

3️⃣ Step 3: ffmpeg でフレーズごとに動画分割 健人

Whisperから得たタイムスタンプで動画を正確にカット。同時にMOV → MP4変換。

pythonfrom config import CLIP_PADDING_SEC, OUTPUT_FORMAT def split_video(video_path: str, phrases: list[dict], output_dir: str): """タイムスタンプに基づいて動画をフレーズごとに分割""" os.makedirs(output_dir, exist_ok=True) for i, phrase in enumerate(phrases): start = max(0, phrase["start"] - CLIP_PADDING_SEC) end = phrase["end"] + CLIP_PADDING_SEC duration = end - start output_file = os.path.join(output_dir, f"clip_{i:03d}.{OUTPUT_FORMAT}") subprocess.run([ "ffmpeg", "-i", video_path, "-ss", str(start), "-t", str(duration), "-c:v", "libx264", # H.264（軽量化） "-crf", "23", # 品質 "-preset", "fast", "-c:a", "aac", "-b:a", "128k", "-movflags", "+faststart", # Web再生向け最適化 "-y", output_file ], check=True, capture_output=True) phrase["video_file"] = f"clip_{i:03d}.{OUTPUT_FORMAT}" return phrases
✅ MOV → MP4 変換効果
ファイルサイズが 1/5〜1/10 に削減。App Storeの200MB OTA制限回避にも有効。

4️⃣ Step 4: Claude API で解説・クイズ・例文を自動生成 健人

プロンプト: pipeline/prompts/generate_lesson.txt

promptあなたは英語教育の専門家です。以下の英語フレーズについて、 日本人の英語学習者向けにレッスンカードのデータを生成してください。 ## 入力フレーズ {phrase} ## 出力形式（JSONで返してください） { "en": "英語フレーズ（重要な表現を<span class=\"hl\">タグ</span>で囲む）", "enPlain": "英語フレーズ（HTMLタグなし）", "ja": "自然な日本語訳", "category": "カテゴリ（日常会話/旅行/ビジネス/レストラン/空港/ショッピング/自己紹介/感情表現）", "choices": ["正解の日本語訳", "紛らわしい誤訳1", "紛らわしい誤訳2", "紛らわしい誤訳3"], "correctIndex": 0, "explain": "<p>文法・ニュアンス・使われる場面の解説。<b>重要な単語</b>はbold。</p>", "examples": [ {"en": "例文1の英語", "ja": "例文1の日本語訳"}, {"en": "例文2の英語", "ja": "例文2の日本語訳"}, {"en": "例文3の英語", "ja": "例文3の日本語訳"} ] } ## ルール - choicesの正解は必ずindex 0に入れる（システム側でシャッフル） - 誤訳は「ありそうで間違い」なものにする - 解説は中学〜高校レベルの学習者がわかる言葉で - 例文は日常で実際に使えるものを選ぶ - JSONのみ返すこと

コード: 解説生成

pythonimport anthropic from config import ANTHROPIC_API_KEY claude = anthropic.Anthropic(api_key=ANTHROPIC_API_KEY) def generate_lesson_data(phrases: list[dict]) -> list[dict]: """Claude API でフレーズごとに解説・クイズ・例文を生成""" template = load_prompt_template() results = [] for i, phrase in enumerate(phrases): prompt = template.replace("{phrase}", phrase["text"]) response = claude.messages.create( model="claude-sonnet-4-20250514", max_tokens=1024, messages=[{"role": "user", "content": prompt}] ) raw = response.content[0].text.strip() if raw.startswith("```"): raw = raw.split("\n", 1)[1].rsplit("```", 1)[0] try: lesson = json.loads(raw) lesson["video"] = phrase.get("video_file", "") results.append(lesson) except json.JSONDecodeError: print(f" [WARN] JSON解析失敗: {phrase['text'][:40]}...") return results

Claude API が自動生成するもの

項目内容
日本語訳自然な日本語訳
解説文法ポイント・ニュアンス・使用場面（HTML形式）
4択クイズ正解 + 紛らわしい誤答3つ
例文実用的な例文3つ（英語 + 日本語訳）
カテゴリ日常会話 / 旅行 / ビジネス 等を自動判定
ハイライト重要表現にHTMLタグ付与

5️⃣ Step 5: Google Spreadsheet に自動登録 健人

事前準備

  1. Google Cloud Console でプロジェクト作成
  2. Google Sheets API を有効化
  3. サービスアカウント作成 → JSONキーをダウンロード → pipeline/credentials.json に配置
  4. スプレッドシートを作成し、サービスアカウントのメールアドレスに編集権限を付与

スプレッドシートの列構成

ABCDEF〜IJK
ステータス先生ID英語日本語訳カテゴリ選択肢1〜4解説動画ファイル
📌 Masaさんの確認フロー
  1. スプレッドシートを開く
  2. 「未確認」のフレーズを上から確認
  3. 日本語訳・解説がおかしければ修正
  4. OKなら「確認済み」に変更

コード: スプレッドシート連携

pythonfrom google.oauth2.service_account import Credentials from googleapiclient.discovery import build from config import SPREADSHEET_ID, SHEET_NAME def upload_to_spreadsheet(lessons: list[dict], teacher_id: str): creds = Credentials.from_service_account_file( "credentials.json", scopes=["https://www.googleapis.com/auth/spreadsheets"] ) service = build("sheets", "v4", credentials=creds) rows = [] for lesson in lessons: choices = lesson.get("choices", [""]*4) rows.append([ "未確認", teacher_id, lesson.get("enPlain", ""), lesson.get("ja", ""), lesson.get("category", ""), *[choices[i] if i < len(choices) else "" for i in range(4)], lesson.get("explain", ""), lesson.get("video", ""), ]) service.spreadsheets().values().append( spreadsheetId=SPREADSHEET_ID, range=f"{SHEET_NAME}!A:K", valueInputOption="RAW", insertDataOption="INSERT_ROWS", body={"values": rows}, ).execute()

🚧 トラブルシューティング

「フレーズが検出されませんでした」

  • 動画に英語の音声が入っているか確認
  • 音声が小さすぎないか（マイクに近づいて話す）
  • language="en" により日本語のみの区間は正常に除外される

「JSON解析失敗」

  • Claude APIの応答がJSON以外を含んでいる場合に発生
  • プロンプト末尾に「JSONのみ返すこと」を強調
  • 失敗したフレーズは手動でスプレッドシートに追加

動画クリップの前後が切れている

  • config.py の CLIP_PADDING_SEC を 0.3 → 0.5 に増やす

ffmpegエラー

  • brew install ffmpeg でインストール済みか確認
  • ffmpeg -version で動作確認
